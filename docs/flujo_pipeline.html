Flujo del pipeline: de transposones a módulos proteicos

Todos los pasos, qué entra y qué sale de cada uno, cómo se concatenan, y la decisión clave que toma. Cada paso pertenece a uno de los dos mundos — el genómico (contar inserciones) y el proteico (armar módulos para homología) — o al tronco compartido (calidad y coordenadas).

Los dos mundos · qué decide cada uno

Compartido

calidad + coordenadas · pasos 01-02

  • Solo CDS protein_coding
  • Unifica cromosomas + liftOver; excluye tx mal lifteados
  • Proyecta genómico → aa
  • Filtros e-value≤1e-10, coverage≥30
  • Bandera sano (no filtra)

→ tabla de hits (3697)

Genómico

contar inserciones · paso 04

  • Dedup a posiciones (coords+cadena+subfam)
  • Cluster por solape + reglas familia-aware
  • SVA contiene Alu → colapsa
  • Módulos distintos → loci separados
  • Representante = menor e-value

→ loci = inserciones (3016)

Proteico

módulos para homología · paso 05

  • Extrae por posición × isoforma (crudo)
  • Dedup por contención dentro del locus
  • Conserva secuencias distintas (isoformas)
  • Nunca dedup entre loci (= señal)

→ dominios distintos (3101)
